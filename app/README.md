# Video Editor App

基于 Vue 3 + TypeScript 的视频编辑器应用

## 功能特性

- 视频播放控制（播放/暂停/快进/快退）
- 可调节的时间轴界面
- **键盘快捷键支持**
  - `空格`: 播放/暂停
  - `←`: 快退 10 秒
  - `→`: 快进 10 秒
  - `F`: 全屏/退出全屏
  - `Esc`: 退出全屏
  - `?`: 显示/隐藏快捷键帮助
- **全屏播放功能**
- **专业 Logo 设计**
  - 渐变色彩和动画效果
  - 多尺寸支持 (sm/md/lg)
  - 响应式交互动画
  - 欢迎页面展示
